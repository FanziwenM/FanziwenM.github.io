Maya中Python常用命令及参数

发表于 2019-05-07 | 更新于 2019-06-02 | 分类于 Notes

General

ls()

列出选择

ls命令，会返回场景中（或选择的）物体名称。最常见的用法就是通过名称或通过类型来过滤或者匹配对象。
默认情况下ls会匹配场景中所有的物体，但是它也可以过滤或者列出选择中的物体，当我们使用selection参数时。

ls(sl = True)

经常结合select()使用。

例如：

select()

选择

Flags：

  • add(add)
  • clear(cl)

bool
添加选择

bool
取消选择

delete()

删除模型，属性，历史记录等

delete()命令在空列表的情况下也不会失败。默认删除选择的模型。

也可以指定删除特定的物体；

Flags:

  • all(all)
  • constructionHistory (ch)

bool
删除所有指定类型，与下面的参数关联

int
当all=True时清除场景中的所有历史记录。
默认清除选择物体的历史记录

group()

将指定物体成组，返回组的名字。

Flags:

  • empty(em)
  • name(n)
  • parent(p)

bool
创建一个空组

unicode
组的名字

unicode
讲组放入指定父级

parent()

通过此命令，移动物体到一个新的组下面。

与上述group()中的”parent”不同。group()指定父级的时候会新建一个组。
比如通过group("cube",p="group1")指定”cube”到组里，如果组”group1”存在，则该命令会在”group1”下面新建一个组”group2”，然后才是物件”cube”

Flags：

  • relative (r)
  • world (w)

bool
如果为True，当父节点有Transform信息，当物体加到父节点下时会产生相应的Transform

bool
相当于Maya快捷键Shift+P，把物体移出组，到世界位置下。

align()

将模型以x,y或者z轴对齐

Flags：

  • alignToLead(atl)
  • xAxis(x)
  • yAxis(y)
  • zAxis(z)

bool
当设置为True,模型将以跟最后一个选择的模型的最小，中心，最大值对齐。
当设置为False,模型将以所有模型中的最小，中心，最大值对齐。

unicode
可以通过'min' 'mid' 'max' 'dist' 'stack'设置对齐类型：
最小，中心最大，分散，堆叠

polyUVSet()

Flags:

  • copy(cp)
  • create(cr)
  • currentUVSet(cuv)
  • delete(d)
  • rename(rn)
  • uvSet(uvs)
  • newUVSet(nuv)

bool
复制当前UV到一个新的UVSet

bool
创建一个新的UVset

bool
跳转到指定的UVset
例如：
polyUVSet(currentUVSet=True, uvSet="uv02")
跳转到uv02作为当前uv

bool
删除当前，或者指定UVset

bool
重命名当前，或者指定UVset

unicode
创建，复制，删除等操作。当需要指定UVset时，用该命令指定特定UVset

unicode
用于重命名时指定新的UVset命名

copy create delet等操作默认操作当前UVset，可以用uvSet加uv名称来指定。

可通过访问UVSet的索引，来得到各个uv的属性。

  • userDefined(ud)

float3

int
以序列的方式指定用户自定的颜色给选中的模型。序列范围(1-8)

createNode

bool
设置为True时，创建后不选择

button()

设置按钮

创建一个按钮控件，显示名字备注，在用户点击时执行命令

Flags:

  • actOnPress(aop)
  • annotation(ann)
  • backgroundColor(bgc)
  • command(c)
  • enable(en)
  • label(l)
  • width(w)

bool
设置为True时，命令将在按钮被点击时就执行，默认松开执行。

unicode
设置备注，鼠标悬停时会提示

float3
设置按钮背景颜色(r,g,b)，RGB值范围到0-1

script
当鼠标点击，执行命令

bool
默认设置为True，按钮为启动状态，设置False时按钮变成灰色，并且不可点击。

unicode
设置按钮名称

int
设置按钮的宽度

unicode,int

推荐使用both

NameSpace

命名时在*string*前加[namespace]:

| *

Maya中特殊字符
|是父子级的分割。例如group1|pCube1
*类似正则表达式中的匹配所有字符集

例如p*就表示所有以p开头的物体，*1则表示所有以1结尾的物体。当然也可以结合上面提到的|
