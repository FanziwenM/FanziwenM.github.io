pm.select()

选择

Flags：

  • add(add)
  • clear(cl)

bool
添加选择

bool
取消选择

pm.ls()

列出选择

pm.ls(sl = True)

经常结合pm.select()使用。

例如：

pm.delete()

删除模型，属性，历史记录等

delete()命令在空列表的情况下也不会失败。默认删除选择的模型。

也可以指定删除特定的物体；

Flags:

  • all(all)
  • constructionHistory (ch)

bool
删除所有指定类型，与下面的参数关联

int
清除场景中的历史记录

pm.group()

将指定物体成组，返回组的名字。

Flags:

  • empty(em)
  • name(n)
  • parent(p)

bool
创建一个空组

unicode
组的名字

unicode
讲组放入指定父级

例如：

pm.move()

移动

Flags：

  • absolute (a)
  • relative (r)

bool
以绝对位置移动

bool
以相对位置移动

pm.align()

将模型以x,y或者z轴对齐

Flags：

  • alignToLead(atl)
  • xAxis(x)
  • yAxis(y)
  • zAxis(z)

bool
当设置为True,模型将以跟最后一个选择的模型的最小，中心，最大值对齐。
当设置为False,模型将以所有模型中的最小，中心，最大值对齐。

unicode
可以通过'min' 'mid' 'max' 'dist' 'stack'设置对齐类型：
最小，中心最大，分散，堆叠

unicode
可以通过'min' 'mid' 'max' 'dist' 'stack'设置对齐类型：
最小，中心最大，分散，堆叠

unicode
可以通过'min' 'mid' 'max' 'dist' 'stack'设置对齐类型：
最小，中心最大，分散，堆叠

pm.color

设置线框颜色

Flags:

  • rgbColor(rgb)
  • userDefined(ud)

float3
指定RBG颜色给选择的模型

int
以序列的方式指定用户自定的颜色给选中的模型。序列范围(1-8)

pm.createNode

创建Node

Flags:

  • name(n)
  • parent(p)
  • shared(s)
  • skipSelect(ss)

unicode
设置名字

unicode
指定其父级之下

bool
设置为True时，Node只会在不存在的情况下创建

bool
设置为True时，创建后不选择

Trick

pm.PopupError()

报错弹窗

pm.isTrue()

检测选择

pm.objExists()

NameSpace

命名时在*string*前加[namespace]:

| *

Maya中特殊字符
|是父子级的分割。例如group1|pCube1
*类似正则表达式中的匹配所有字符集

例如p*就表示所有以p开头的物体，*1则表示所有以1结尾的物体。当然也可以结合上面提到的|
