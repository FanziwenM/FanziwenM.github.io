#include

头文件

#include包括两种形式：

  • #include< >
  • #include" "

#include< > 引用的是编译器的类库路径里面的头文件。
#include" " 引用的是你程序目录的相对路径中的头文件。

有些C头文件被转换为C++头文件，这些文件被重新命名，去掉了扩展名h(使之成为C++风格的名称),但是在文件名称前加上前缀c（表示来自C语言),例如C语言的数学函数#include

const

const T

如我们所知，const是用来定义一个常量，一旦声明值将不能再被改变

const声明变量的时候必须给一个初始值

const T*

指向常量的指针， 不能改变指向对象的值，但是可以改变指针本身的值，即可以改变对象的地址。
