Command与Batch常用命令

发表于 2019-05-06 更新于 2019-07-04 分类于 Tech

📃Command

通用操作

$ help

可查看该命令的介绍、参数列表与使用方法

$ cls

清屏

$ tree

显示目录结构

Ctrl + Z

中断命令

创建操作

$ md [path]

创建目录

$ md a,b,c
在当前目录下创建文件夹a，文件夹b，文件夹c

$ md a\b\c
在当前目录下创建文件a，在文件夹a下创建b，在文件夹b下创建c

cd 切换目录操作

$ cd [path]

切换目录

$ cd /d [drive:][path]

跨盘切换目录

$ cd..

返回上一级

tree

显示目录结构

/f 目录结构显示所有文件

dir 显示目录操作

$ dir

显示当前目录中的文件夹于文件信息

$ dir /b

只显示当前目录中的子文件夹于文件名字

$ dir /ad

只显示当前目录中的文件夹

$ dir /a-d

只显示当前目录中的文件

$ dir /s

递归显示当前目录中的内容

$ dir a*

显示当前目录下以a开头的文件和文件夹信息

删除操作

$ rm [file-name]

只能删除文件

$ rm -a [file-name]

删除包含文件的文件夹

$ rm -rf

强制递归删除

⚙️Batch

相对路径

%~dp0表示批处理文件所在目录

cd /d %~dp0可以切换到批处理文件所在目录

拷贝

copy拷贝单个文件

xcopy拷贝一组文件

Flags:

Parameter Description
/EXCLUDE:[file1] 可以通多读取一个文件来排除特定的文件类型。例如我创建一个txt文件exclude.txt写入”.obj”。然后在批处理中xcopy /EXCLUDE:exclude.txt就可以排除obj文件
/c 即使有错误，也继续复制
/y 禁止提示以确认改写一个现存目标文件
/s 复制目录和子目录，除了空的
/e 复制目录和子目录，包括空的
/u 只复制已经存在于目标中的文件
/q 复制时不显示文件名
/f 复制时显示完整的源和目标文件名
