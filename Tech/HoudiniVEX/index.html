Adds a primitive to the geometry.

使用点序号指定的点来创建图元

使用一组点序号数组指定的点来创建图元

Returns the list of points on a primitive.

Math

数学相关

avg()

Returns the average value of the input(s)

sin(), cos()

as you’d expect, but in radians.

rint()

Rounds the number to the closest whole number.

radians()

convert a number from degrees to radians: foo = radians(90);

length()

measure the length of a vector. Eg, measure the distance of a point from the origin: dist = length(@P);

distance()

measure the distance between two points: dist = distance(@P, v@mypoint);

ident()

Returns an identity matrix.

Interpolation

插补

fit()

take a number between 2 values, fit it between 2 other values, usually 0-1. Eg, remap @u from range -5,20 to 0-1: foo = fit(@u, -5, 20, 0, 1);

将一个区间里的值，映射到另一区间里。
例如:将100个点，根据点序号(0-99)，填充渐变颜色(0-1）

fit01()

Takes the value in the range (0, 1) and shifts it to the corresponding value in a new range.

Rotate with a Matrix

在Houdini中矩阵旋转没有听上去的难，只要简单的三步：

  • 创建矩阵(Create a matrix)
  • 旋转矩阵(Rotate the matrix)
  • 将矩阵应用于模型(Apply the matrix to geometry)

听上去第二步旋转(Rotate)矩阵似乎简化了，这一步在数学运算中虽然没有那么简单，但是在VEX中，有一个函数rotate()可以直接旋转矩阵
而第三步的应用，则是直接将模型和变化矩阵相乘即可
