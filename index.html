🌏基础介绍

VanTools期初是为了方便自身工作需要，用业余时间通过Python编写的一些小工具。
其目的是为了提高工作效率的同时，锻炼自己编码能力。
随着功能的逐渐增加，更新和完善。工具被除了我之外更多人使用，所以写了这篇说明。

下载安装

Github:https://github.com/FanziwenM/VanTools
Colne 或 下载解压后，运行.bat文件即可(需先关闭Maya)

.bat文件会将工具的代码，图标和Shelvs拷贝到Maya的prefs(路径:Documents/maya/@VERSION/prefs),工具是在Maya2018下编写的，所以这里的路径中的@VERSION是2018。其他Maya版本请改成对应版本。但是其他版本不确定工具是否正常运行

发表于 2019-06-20 更新于 2020-11-14 分类于 Tech

💾Module

模块

随着程序变得越来越长，你或许会想把它拆分成几个文件，以方便维护。你亦或想在不同的程序中使用一个便捷的函数， 而不必把这个函数复制到每一个程序中去。
为支持这些Python有一种方法可以把定义放在一个文件里，并在脚本或解释器的交互式实例中使用它们。这样的文件被称作模块(Module) ；模块(Module)中的定义可以导入到其它模块(Module)或者主(Main)模块

Name

全局变量__name__

每一个.py文件都可以视为一个模块(Module)。而且当一个模块被导入时，可以通过全局变量__name__获取名字(字符串)。
