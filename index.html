UE4材质基础

发表于 2019-06-02 | 更新于 2019-06-13 | 分类于 Docs

🌏基础介绍

材质是用了一种特别的编程语言：高阶着色器语言(High Level Shading Language)简称HLSL 创建的。HLSL可以使材质直接和图形硬件通信，从而使美术和程序员直观的观察控制屏幕显示内容。Unreal Engine 4 (UE4)中用来创建材质球的材质节点包含着和HLSL一样的代码块，来实现和这些代码块相同的效果。

主材质节点(The Main Material Node)

在Content Browser中右键创建一个材质

双击我们新建的材质，进入材质编辑窗口。

主材质节点(The Main Material Node)可以被认为材质制作的最后一步。将材质节点任意组合后，输入给主节点，就会有相应的效果在材质球编译后显示。
