常用参数

方向参数

  • TB 从上到下

  • BT 从下到上

  • RL 从右到左

  • LR 从左到右

  • TD 同TB

箭头参数

  • A –> B A带箭头指向B
  • A — B A不带箭头指向B

  • A -.- B A用虚线指向B

  • A -.-> B A用带箭头的虚线指向B

  • A ==> B A用加粗的箭头指向B

  • A – 描述 — B A不带箭头指向B并在中间加上文字描述

  • A – 描述 –> B A带箭头指向B并在中间加上文字描述

  • A -. 描述 .-> B A用带箭头的虚线指向B并在中间加上文字描述

  • A == 描述 ==> B A用加粗的箭头指向B并在中间加上文字描述

图形参数

  • id + [文字描述]矩形

  • id + (文字描述)圆角矩形

  • id + >文字描述]不对称的矩形

  • id + {文字描述}菱形
  • id + ((文字描述))圆形
